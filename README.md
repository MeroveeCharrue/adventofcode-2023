# Advent of Code - 2023

Another year, another [AdventOfCode](https://adventofcode.com/2023) :)

This time, let's play with Symfony Console component !

## Solve it all!

This requires PHP ^8.3 and Composer.

```shell
# Install dependencies.
composer install

# Enjoy the ride!
./advent
```

## Tests

```shell
./vendor/bin/phpunit
```
